# liaosuiyang.github.io

Personal academic website for Suiyang Liao, served via GitHub Pages as static HTML/CSS.

## Structure

- `index.html` — the site
- `assets/css/style.css` — styles
- `assets/img/` — photos, publication figures, research figures

## Local preview

```
python3 -m http.server 8000
```

then open `http://localhost:8000`.
